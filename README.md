# 리그오브레전드 천상계 매치 데이터 분석 및 웹페이지 개발

## 프로젝트 설명
이 프로젝트는 리그오브레전드 데이터 EDA 및 승리 요인 분석의 후속 프로젝트로, 챔피언 분석, 라인별 분석 내용을 담은 Tableau 대시보드를 만들고, 이를 웹사이트에 배포하는 것을 목표로 합니다. 천상계 매치 데이터를 활용하여 보다 심층적인 분석을 제공하며, 사용자가 쉽게 이해할 수 있도록 시각화된 대시보드를 제공합니다.

## 사용된 데이터
- **천상계 매치 데이터:** 라이엇 오픈 API에서 가져온 데이터
    - 매치 데이터는 천상계 (마스터 ~ 챌린저) 각 티어별로 랜덤 샘플링 하여 10%씩 추출해 가져왔습니다. (예: 챌린저 300명 -> 30명 랜덤 샘플링)
- **챔피언 데이터:** DataDragon에서 가져온 데이터
- **아이템 데이터:** DataDragon에서 가져온 데이터
- **소환사 주문 데이터:** DataDragon에서 가져온 데이터

## 데이터 분석 과정
1. **데이터 수집:** 라이엇 오픈 API를 사용하여 천상계 매치 데이터, 아이템 데이터, 챔피언 데이터, 소환사 주문 데이터를 수집.
2. **데이터 전처리:** Python을 사용하여 수집한 데이터를 전처리.
3. **데이터베이스 구축:** 전처리된 데이터를 구글 빅쿼리를 사용하여 데이터베이스로 구축.
4. **Tableau 연결:** 빅쿼리에 업로드된 데이터를 Tableau와 연결하여 시각화.
5. **대시보드 개발:** 
    - 챔피언 별 분석 내용을 담은 대시보드 개발
    - 라인별 지표 분석 내용을 담은 대시보드 개발
6. **웹페이지 개발:** Bootstrap을 사용하여 웹페이지를 개발.
7. **배포:** 대시보드와 전 프로젝트 리포트를 Netlify를 통해 배포.
